# Easy Parachute Wind Tunnel for Kids

This is the **simple version** of the project.

It is made for a kid with adult help.

If you want the full detailed version, see `README-advanced.md`.

## What You Are Building

You will build:

1. a simple wind tunnel that blows air upward
2. a few small parachutes
3. a test area where you can see which parachute works best

The goal is to answer questions like:

- Which parachute shape is the most stable?
- Which parachute can hold the most weight?
- Which parachute opens the easiest?

## Important Safety

- Have an adult help with cutting cardboard, plastic, wood, or screen.
- Keep fingers away from the fan.
- Put a screen or guard over the fan before using it.
- Tie back long hair.
- Do not use heavy objects as payloads.
- Start with tiny weights like washers or paper clips.

## Super Simple Idea

This project uses a **box fan** pointing upward.

Above the fan, you build a cardboard tower with:

1. a fan section
2. a calming section with screen
3. a clear test window

That is enough to compare parachutes.

## Materials

You can get most of this at home, on Amazon, or at Home Depot.

### Wind Tunnel Materials

- `20 in box fan`
- `2 large cardboard boxes`
- `clear plastic sheet` or a clear storage bag
- `window screen` or mesh laundry bag material
- `duct tape`
- `packing tape`
- `hot glue` or school glue
- `scissors`
- `box cutter` for adult use
- `ruler`
- `marker`

### Parachute Materials

- `trash bag plastic` or thin plastic tablecloth
- `yarn`, `string`, or fishing line
- `tape`
- `paper clips`
- `washers` or small nuts for weights

### Nice Extras

- `binder clips`
- `smartphone` for slow-motion video
- `small ruler` taped to the side of the tunnel

## Easy Size Guide

Use these simple sizes:

- fan: `20 in`
- tunnel width: about `16 in to 20 in`
- tunnel height above fan: about `24 in to 36 in`
- parachute size: `4 in to 6 in` across

Keep the parachutes small. Small parachutes work much better in a home fan tunnel.

## How to Build the Wind Tunnel

### Step 1: Set Up the Fan

1. Put the box fan on the floor or a sturdy table.
2. Make sure it blows **upward**.
3. Tape a screen or guard over any opening where fingers could reach the blades.

### Step 2: Make the Tower

1. Cut the bottom out of a cardboard box.
2. Set that box on top of the fan.
3. Tape the box to the fan edges so air mostly goes into the box.
4. If needed, stack a second box on top to make the tower taller.
5. Tape the boxes together.

You now have a simple wind tunnel body.

### Step 3: Add a Screen Layer

1. Cut a piece of window screen or mesh.
2. Tape it inside the tower a little above the fan.
3. If you have extra mesh, add a second layer about `1 in` above the first.

This helps make the air less messy.

### Step 4: Make a Clear Window

1. Cut a square window in one side of the upper box.
2. Tape clear plastic over the hole.
3. Tape a ruler next to the window.

Now you can watch the parachute move.

### Step 5: Add a Hanging Point

1. Poke a tiny hole at the top of the tunnel.
2. Run a string through it.
3. Tape it in place.
4. Add a small loop or paper clip so you can hang parachutes from it.

## How to Make Easy Parachutes

Make at least 3 different parachutes.

Try to change only one thing at a time.

### Parachute 1: Round

1. Draw a circle on plastic.
2. Cut it out.
3. Add 6 or 8 small pieces of tape around the edge.
4. Punch or poke tiny holes through the tape spots.
5. Tie equal strings to each hole.
6. Tie all strings together at the bottom.
7. Add one paper clip or washer.

### Parachute 2: Round With a Hole

1. Make the same round parachute.
2. Cut a small hole in the middle.

This center hole is called a spill hole.

### Parachute 3: Square

1. Cut a square piece of plastic.
2. Add one string at each corner.
3. Tie the strings together.
4. Add the same weight as the other parachutes.

### Parachute 4: Cross Shape

1. Draw a plus-sign shape.
2. Cut it out.
3. Add strings at the outer corners.
4. Tie the strings together.
5. Add the same weight.

## Rule for Fair Testing

When you test parachutes, keep most things the same:

- same fan
- same tunnel
- same string length
- same weight

Only change **one thing** at a time:

- shape
- size
- center hole

## How to Test

### Test 1: Does It Open?

1. Turn the fan on low.
2. Hang the parachute in the tunnel.
3. Watch what happens.

Ask:

- Does it open all the way?
- Does it twist?
- Does it fold up?

### Test 2: Which One Is Most Stable?

1. Let the parachute float in the air.
2. Watch for `20 seconds`.

Ask:

- Does it stay in the middle?
- Does it spin?
- Does it wobble a lot?

### Test 3: Which One Holds More Weight?

1. Start with one washer or one paper clip.
2. Test the parachute.
3. Add one more small weight.
4. Test again.
5. Stop when the parachute can no longer float well.

The one that holds the most weight is making more drag in your tunnel.

## Good Starter Plan

Build these first:

1. `4 in` round parachute
2. `4 in` round parachute with a small center hole
3. `4 in` square parachute

Then try:

1. bigger parachutes
2. longer strings
3. different materials

## Easy Tips

- If the parachute spins, check whether the strings are the same length.
- If it keeps collapsing, try a small hole in the middle.
- If it hits the side wall, make the parachute smaller.
- If it does not lift, use a lighter weight.
- If the air is too messy, add another screen layer inside the tunnel.
